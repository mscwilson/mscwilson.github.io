About this site

This page is created using Jekyll and the Minimal Mistakes theme, which provides all the necessary structure, configuration, and styling. This was perfect for me at the start of 2021, before I knew anything about web development. It was initially a platform for my blog, which I started so I could share my excitement at all the new things I was discovering during my bootcamp.

Jekyll is a static site generator and is designed with blogging in mind, as all that's needed to write a new post is to create a new Markdown file. It seemed like it would be more fun and educational to make my own site than posting on Medium or WordPress. This site is hosted with GitHub Pages, a free GitHub-based hosting service based on Jekyll.

I'm slowly working on personalising the code. My next goal is to add a portfolio page to share all the cool stuff I've made. In the meantime, check out my GitHub.
